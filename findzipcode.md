# 개인정보처리방침

**새주소 검색** 앱(이하 "앱")을 이용해 주셔서 감사합니다. 본 개인정보처리방침은 앱 이용 과정에서 개인정보가 어떻게 수집, 이용, 관리되는지에 대해 안내드립니다.

## 1. 개인정보의 처리 목적

본 앱은 다음과 같은 목적으로 개인정보를 처리합니다:
- 한국 우편주소 검색 서비스 제공
- 즐겨찾기 기능을 통한 사용자 편의성 향상
- 앱 서비스 품질 개선 및 통계 분석

## 2. 수집하는 개인정보 항목

### 2.1 직접 수집 정보
- **즐겨찾기 데이터**: 사용자가 저장한 주소 정보 및 메모 (사용자 기기)

### 2.2 자동 수집 정보
- **앱 사용 통계**: Firebase Analytics를 통한 앱 사용 패턴 (익명화)
- **오류 보고서**: Firebase Crashlytics를 통한 앱 안정성 개선 (익명화)
- **광고 식별자**: Google AdMob을 통한 맞춤형 광고 제공
